The Open Telemetry Unit
Open source bare-metal programming for ARM Cortex-M series 

This program is made to reproduce and Industrial Internet Of Thing Telemetry Unit, it will cover :
- ARM9 32Bits, 400mhz 
- Real-time clock
- DC to AC
- DI Counters inputs
- AC to DC
- RS232 
- RS485
- Ethernet  
- USB 
- I2C
- UART
- Watchdog circuitry in C
